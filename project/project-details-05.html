2048 [Game]

C++ Game

Project 2048 is 16 cells–1 grid C++ game. It is very small sized game with clear user interface. There is only one rule, join the numbers and get to the ‘2048’ tile. There is no time limitation. A great 'undo' feature is also available, so that it does not spoil your strategy to win the game. It also keeps record of your high score. It will give you a moral boost each time you play the game.
