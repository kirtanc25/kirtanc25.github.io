Gujarati Handwritten Character Recognition

Machine Learning

Looking for Something Interesting?

It's a mobile application that takes scanned gujarati handwritten image as input and extracts characters (alphabets, digits, symbols) from it. The image can be of a handwritten gujarati document or printed document.

  • Details
  • May, 2020
  • handwritten-character-recognition, gujarati-language, gujarati-dataset, image-processing
  • More Info

This mobile application can be used by anyone to recognize handwritten gujarati characters and convert it to a readable format. The mobile application recognizes characters with an accuracy of 90%. The mobile application depends on the scanning quality of camera, if quality is good then it can able to recognize the text easily and with greater accuracy.
